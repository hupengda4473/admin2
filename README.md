#### 说明
项目在Light Year Admin的基础上整理修改而来，用到了开源项目[Bootstrap-Multitabs](https://gitee.com/edwinhuish/multi-tabs)来实现多标签页，稍微做了一些修改。在这感谢该开源项目的小伙伴。

- 该项目是景电智慧灌区信息化管理系统的ifame版本。
- 本项目主要使用白色主题
